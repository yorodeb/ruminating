# 🧪 The Curious Cabinet of Code

Welcome to a repository that refuses to sit still.

This is not your typical polished, single-purpose project. Instead, think of it as a digital junk drawer—except every “junk” item has a story, a lesson, or at least a strange personality.

---

## 🌀 What Lives Here?

### 🧩 Weird & Wacky Programs
A collection of experiments, half-serious ideas, and “what happens if I…” moments.

- Programs that solve problems nobody asked about  
- Code written to explore odd edge cases  
- Mini-projects that started as jokes and spiraled into something real  
- Possibly useful. Possibly cursed. Definitely interesting.

> "What I can't build, I don't understand.."

## 🧭 Philosophy of This Repo

This repository follows a simple rule:

> If it sparks curiosity, it belongs here.

Not everything is perfect. Not everything is finished.  
But everything here represents exploration.
